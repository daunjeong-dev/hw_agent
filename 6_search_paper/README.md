# 논문 탐사 에이전트 (Paper Exploration Agent)

## 목적

AI 기술은 빠르게 변화하며 새로운 논문가 지속적으로 발표된다.  
새로운 기술이 실제로 검토할 가치가 있는지 판단하고, 핵심 내용을 이해하는 과정에는 많은 시간과 노력이 필요하다.

이 에이전트는 최근 AI 논문를 탐색하고, 질문 기반으로 논문를 깊이 이해하도록 돕는다.

---

## 핵심 기능

1. **최근 AI 논문 탐색**
   - 사용자 조건에 맞는 최신 논문 검색
   - 추천도 및 메타데이터 기반 정렬

2. **논문 분석 및 질문**
   - 논문 내용 요약
   - 사용자의 사고를 유도하는 질문 생성
   - 답변에 따라 추가 질문 및 피드백 제공

3. **내용 정리 및 저장**
   - 논문 핵심 내용 정리
   - 학습 결과 저장(아직 구현 안됌)

---

## 그래프 구조

```text
START
  │
  ▼
parse_query
  │
  ▼
fetch_and_score
  │
  ├──────────────► summarize_paper (병렬)
  │                    │
  └────────────────────┘
           │
           ▼
user_select_paper
           │
           ▼
assumption_checker
           │
     ┌─────┴───────────────┐
     │                     │
     ▼                     ▼
user_select_paper   assumption_checker
     │                     │
     └──────────┬──────────┘
                ▼
               END
```

---

## 노드 설명

| Node | 역할 |
|------|------|
| `parse_query` | 사용자 검색 조건 추출 |
| `fetch_and_score` | 논문 검색 및 추천도 계산 |
| `summarize_paper` | 논문 요약 및 핵심 내용 생성 |
| `user_select_paper` | 사용자가 탐색할 논문 선택 |
| `assumption_checker` | 질문을 통해 논문를 깊이 이해하도록 지원 |
